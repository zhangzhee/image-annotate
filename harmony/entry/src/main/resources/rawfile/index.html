图片标注 · 使用帮助

      

        

          
快速开始

          
    
            
  • 点工具栏「导入图片」选择本地文件，即可开始标注。
    • 
            
  • 从左侧工具栏选一种标注工具，在图上拖拽绘制；用「选择」工具可点选、移动、修改样式。
    • 
            
  • 标注完成后点「导出 PNG」把带标注的图下载到本地（可先选分辨率与文件夹）；「复制」复制到剪贴板。
    • 
          

        

        

          
工具（左侧栏）

          
    
            
  • 选择：点选 / 移动 / 编辑已有标注。
    • 
            
  • 裁剪：框选需要保留的画面区域。
    • 
            
  • 框选 / 箭头 / 画笔 / 高亮 / 文本 / 编号 / 马赛克：在图上绘制对应标注。
    • 
            
  • 水印：文字或 Logo 水印，支持平铺 / 居中 / 单角、角度与透明度，随导出一起输出。
    •

标注列表

          
    
            
  • 右侧列表展示所有标注，点击可选中并在上方编辑样式。
    • 
            
  • 每行「隐 / 锁 / 删」可分别切换显示、锁定、删除。
    • 
            
  • 顶部搜索框可过滤标注文字 / 名称 / 类型。
    • 
          

        

        

          
小提示

          
    
            
  • 选择工具下，右侧面板会显示图片信息（长宽 / 分辨率 / 文件大小 / 标注数量等）。
    • 
            
  • 在画布中按 Ctrl/⌘ + V 可把系统剪贴板里的图片直接粘贴进来，作为可移动 / 缩放 / 删除的叠加图层。
    • 
            
  • 遇到异常时，界面会显示错误提示，可在浏览器控制台查看详细信息。
    • 
            
  • 标注自动存在本地草稿（localStorage），下次打开同一图片可继续编辑。
    •
